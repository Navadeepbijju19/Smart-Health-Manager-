 
Smart Health Manager is a web-based healthcare management system designed to simplify and digitalize the management of hospitals, clinics, doctors, patients, appointments, and medical records. The system provides a centralized platform where patients can register, book appointments, access prescriptions, and view their medical history, while healthcare providers can efficiently manage patient information and daily operations.The primary objective of the project is to reduce paperwork, improve the accuracy of medical records, and enhance communication between patients, doctors, and administrators. By automating routine healthcare processes, the system minimizes manual errors, saves time, and improves the overall quality of healthcare services.The application includes secure user authentication, role-based access control, appointment scheduling, doctor management, patient record management, prescription generation, billing, notifications, and an admin dashboard for monitoring the entire system. It also supports real-time data management and provides an intuitive interface for users with different roles.
